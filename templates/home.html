Create Twitter campaigns that target US legislators

Mention bin allows anyone to create a campaign page, that lets people target their legislators with a custom message

  • Get Congress's attention over Twitter
  • Free and no registration required
  • Simply click "Create Campaign" in the top right
  • Set a password if you would like to edit it later

Allow people to instantly tweet their representatives

Check out this fun and simple campaign example where you can thank your representatives

Open Source

Mention Bin is an open source project from Taskforce.is.
